# Mandelbrot set visualizer

This is a simple [Mandelbrot set](https://en.wikipedia.org/wiki/Mandelbrot_set)
visualizer developed in C using SDL2 for graphics.

## Dependencies
- [SDL2](https://www.libsdl.org/)

## Building and running
```console
$ make -j $(nproc)
$ ./mandelbrot
```
